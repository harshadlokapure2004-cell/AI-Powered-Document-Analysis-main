
# 📄 AI-Powered Document Analysis

An intelligent document analysis system that extracts, processes, and summarizes text from documents using OCR and AI techniques.

---

## 🚀 Overview

This project allows users to upload documents (images or PDFs) and automatically:

* Extract text using OCR
* Process and clean the extracted content
* Generate meaningful summaries using AI
* Provide a fast and user-friendly interface

It is designed to reduce manual reading effort and improve productivity.

---

## 🧠 Features

* 📷 OCR-based text extraction
* 📝 AI-powered summarization
* ⚡ Fast processing
* 🌐 Web-based interface
* 📂 Support for multiple document formats

---

## 🛠️ Tech Stack

**Frontend:**

* React / Vite
* HTML, CSS, JavaScript

**Backend:**

* Node.js / Express

**AI & Processing:**

* OCR Engine (e.g., Tesseract)
* NLP / AI API

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/harshadlokapure2004-cell/AI-Powered-Document-Analysis.git
cd AI-Powered-Document-Analysis
```

---

## ▶️ Running the Project

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
